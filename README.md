# Git Learning Repository 🧭

This repository was created primarily as a hands-on space to learn and practice Git and GitHub workflows.

It contains important and simple git and github experiments.

---

## 🎯 Purpose

The main goal of this repo is to build comfort with:

- Initializing and managing Git repositories
- Staging and committing changes
- Working with branches
- Connecting local repositories to GitHub
- Pushing and pulling changes
- Understanding version control workflows in real projects

---

## 🧪 What’s inside

This repository may include:

- Test files created during Git practice
- Experimental commits and branch work
- Simple code snippets used for learning
- Notes or logs from Git commands

Nothing here is meant for production use—this is purely a learning sandbox.

---

## 🛠️ Tools used

- Git
- GitHub
- VS Code terminal / Command line
- Windows PowerShell / CMD

---

## 📚 What I learned

Through this repository, I practiced:

- Basic Git commands (`init`, `add`, `commit`, `status`)
- Branching and merging
- Working with remote repositories
- Resolving simple workflow issues
- Understanding how Git tracks changes over time
